✅ AJOUTÉ:

1. Bouton "📱 Ouvrir scrcpy"
   - Sélectionnez un ou plusieurs appareils dans la liste
   - Cliquez sur "Ouvrir scrcpy"
   - Une fenêtre scrcpy s'ouvre pour chaque appareil sélectionné
   - Vous pouvez voir et contrôler le téléphone directement

2. Bouton "🎬 Script sur sélectionnés"
   - Sélectionnez les appareils dans la liste
   - Choisissez votre script dans le menu déroulant
   - Cliquez sur "Script sur sélectionnés"
   - Le script s'exécute uniquement sur les appareils sélectionnés

📋 WORKFLOW COMPLET:

1. 🔌 BRANCHER vos téléphones Android

2. 🚀 LANCER: python run_multi_device.py

3. 🔄 Cliquez "Actualiser" → Voir les appareils

4. ✅ Sélectionnez les appareils → "Configurer sélectionnés"

5. 📱 (OPTIONNEL) Cliquez "Ouvrir scrcpy" pour voir le téléphone

6. 📜 Choisissez votre script dans le menu

7. 🎬 Cliquez "Script sur sélectionnés" pour lancer sur les sélectionnés
   OU "Script sur TOUS" pour lancer sur tous

8. 📊 Cliquez "Statut" pour voir les résultats

💡 ASTUCES:

- Vous pouvez avoir PLUSIEURS fenêtres scrcpy ouvertes
- Chaque script s'exécute en arrière-plan (threading)
- Regardez la console pour suivre l'exécution
- Utilisez "Arrêter tout" pour arrêter tous les scripts
