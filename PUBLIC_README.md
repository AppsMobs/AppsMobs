# AppsMobs - Téléchargement et Installation

## Télécharger
- Dernière version Windows: voir l'onglet "Releases" sur GitHub ou `appsmobs.com/download`.

## Installer
1. Lancez l'installateur `.exe`.
2. Suivez l'assistant d'installation.
3. Ouvrez AppsMobs depuis le menu Démarrer.

## Activer la licence
- Créez un compte et obtenez une clé depuis `appsmobs.com/shop`.
- Renseignez votre email et votre clé dans l'application.

## Vérifier l'intégrité
- Comparez la somme SHA256 dans `SHA256SUMS.txt`.
