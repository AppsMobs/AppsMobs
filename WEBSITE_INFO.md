# 🌐 AppsMobs Website Platform

## Overview

AppsMobs includes a comprehensive web platform available at [appsmobs.com](https://appsmobs.com) that provides essential services for managing your AppsMobs licenses and account.

## Features

### 🔐 User Authentication
- Secure registration and login system
- Email verification
- Password reset functionality
- Session management

### 💳 License Management
- Purchase AppsMobs licenses (Normal, Pro, Team plans)
- View and manage active licenses
- Track license expiration dates
- Automatic license renewal options

### 📊 User Dashboard
- Real-time token balance tracking
- Usage statistics and analytics
- License status overview
- Device management

### 🎁 Referral System
- Generate unique referral codes
- Track referral earnings
- Automatic reward distribution
- Referral statistics

### 💬 AI Support Chat
- Intelligent customer support chatbot
- Contextual responses to common questions
- Pricing and feature inquiries
- Installation help

### 📝 Documentation
- Complete user guides
- API reference documentation
- Code examples and tutorials
- FAQ section

### 🛒 Shop & Payments
- Multiple payment methods (Stripe, PayPal, Binance Pay)
- Discount code system
- Secure checkout process
- Payment history

## Technical Stack

The website platform is built with:
- **Frontend**: React + Vite
- **Backend**: Node.js + Express
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Payments**: Stripe, PayPal, Binance Pay
- **Hosting**: Vercel/Netlify (Frontend) + VPS (Backend)

## Access

- **Website**: [appsmobs.com](https://appsmobs.com)
