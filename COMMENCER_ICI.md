# 🎯 COMMENCEZ ICI !

## 💡 En 30 secondes

### Vous voulez juste LANCER l'application ?

```bash
pip install -r requirements.txt
python run_app.py
```

✅ C'est tout ! L'app se lance.

---

### Vous voulez créer l'EXE pour le CLIENT ?

```bash
pip install pyinstaller
python build_final.bat
```

✅ L'EXE sera dans `dist/BootyBot/BootyBot.exe`

---

### Vous voulez savoir quoi DONNER au client ?

```
📦 ZIP avec:
├── BootyBot.exe (de dist/BootyBot/)
└── INSTRUCTIONS_CLIENT.txt
```

Le client n'a PAS besoin de Python, juste ADB et scrcpy.
